NHPP Regression¶

Proportional-intensity NHPP regression: a time-varying baseline (Duane by default) scaled by the covariate factor exp(Z @ beta).

class ProportionalIntensityNHPP¶
ProportionalIntensityNHPP_.fit(x, Z, i=None, c=None, n=None, t=None, tl=None, tr=None, dist=<surpyval.recurrent.parametric.duane.Duane object>, init=None)¶

Fit the model using the provided data and initial parameters.

Parameters

  • x (array_like) – Input data.

  • Z (array_like) – Covariate matrix.

  • i (array_like, optional) – identity of the item.

  • c (array_like, optional) – Censoring indicators.

  • n (array_like, optional) – Number of events.

  • t (array_like, optional) – (N, 2) array of [left, right] truncation bounds per observation.

  • tl (array_like or scalar, optional) – Left truncation (delayed entry) time per item; the observation of each item begins here. Scalar broadcasts to all items.

  • tr (array_like or scalar, optional) – Right truncation time per item; the observation window closes here, so the baseline intensity is integrated out to tr even without an explicit right-censoring (c=1) row.

  • dist (surpyval.recurrent.regression.NHPPFitter, optional) – The parametric model to use for the hazard rate.

  • init (array_like, optional) – Initial parameter estimates.

Returns

An object containing the results of the fitting process, including parameter estimates.

Return type

ProportionalIntensityModel
