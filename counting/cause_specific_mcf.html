Cause-Specific MCF¶

class surpyval.recurrent.competing_risks.nonparametric.cause_specific_mcf.CauseSpecificMCF¶

Bases: object

Cause-specific Mean Cumulative Function for a recurrent process with competing event types.

The model fits one NonParametricCounting MCF per event type, sharing the at-risk set across causes. Access the per-cause models through self.models[cause] or use the convenience methods below.

classmethod fit(x, i=None, c=None, n=None, e=None, tl=None, tr=None)¶

Fit a cause-specific MCF.

Parameters

  • x (array like) – Event (and censoring) times.

  • i (array like, optional) – Item / subject id for each row. Defaults to a single item.

  • c (array like, optional) – Censoring flag for each row (0 observed, 1 right censored).

  • n (array like, optional) – Count of events at each row. Defaults to 1.

  • e (array like) – Event type (mark) for each row. None for censored rows.

  • tl (array like or scalar, optional) – Left-truncation (delayed-entry) time per item. The at-risk set is shared across causes, so a delayed entry shrinks the risk set for every cause until the item enters at tl.

  • tr (array like or scalar, optional) – Right-truncation time per item.

Return type

CauseSpecificMCF
