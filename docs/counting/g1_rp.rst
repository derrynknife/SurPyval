Generalized One Renewal Process
===============================

The G1 renewal process (Lam's geometric process, reparameterised): the ``j``-th inter-arrival is the base lifetime scaled by ``(1 + q)**j``. Returns a :doc:`Renewal Model <renewal_model>`.
