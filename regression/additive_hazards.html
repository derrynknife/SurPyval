Fit the Lin & Ying additive hazards model.

Parameters

  • x (array-like) – The observed event/censoring times.

  • Z (array-like) – The covariate matrix (one row per observation).

  • c (array-like, optional) – Censoring flags: 0 observed (event), 1 right-censored. Defaults to all observed.

  • n (array-like, optional) – Multiplicity (counts) for each row. Defaults to 1 each.

Returns

The fitted model, carrying beta, standard errors, the coefficient covariance, p-values, and the baseline hazard.
